### docker 命令

docker ps -a 检查全部的容器； 	

docker start [ ID ] 启动对应机器；

docker images 查看当前所有镜像；

docker rm[ ID ] 移除对应的镜像；



安装 mysql 镜像：

 docker run --name mysql  -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root -d mysql:latest
